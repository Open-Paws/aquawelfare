# AquaWelFare — Agent Overview

AquaWelFare is an Open Paws advocacy tool for aquatic animal welfare data visualization and policy analysis.

It covers welfare gap data for 40+ countries and 35+ farmed aquatic species, with an interactive policy intervention simulator that projects how specific legislative or certification changes would reduce the number of animals living without welfare protections.

This tool operates in the Public Campaigns bounded context — it is public-facing, educational, and advocacy-oriented. It does not handle investigation data or sensitive activist information.
